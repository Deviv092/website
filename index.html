Projects

Rock-Paper-Scissor Game

Description: A simple web app where players compete against the computer in the classic Rock-Paper-Scissors game.

Key Features:

  • User Input: Choose between rock, paper, or scissors.
  • Game Logic: The computer randomly picks, and the winner is determined by standard rules.
  • Result Display: Shows player and computer choices, and declares the outcome (win, lose, tie).
  • Score Tracking: Optional feature to track wins, losses, and ties.

Tech Stack: HTML, CSS, JavaScript.

Currency Converter Web App

Description: A real-time currency converter that allows users to convert between different currencies using live exchange rates.

Key Features:

  • User Input: Enter amount and select source/target currencies.
  • Real-Time Conversion: Uses live exchange rates from external APIs.
  • Output Display: Shows the converted amount.
  • Optional Features: Save favorite currencies and view historical rates.

  • Win/Tie Conditions: Automatically detects win, tie, or ongoing game.
  • Restart Option: Reset the board for a new round.
  • Optional AI: Play against a basic AI opponent.

Tech Stack: HTML, CSS, JavaScript.
